> 基本

:::demo 最基础的用法，通过设置`to`属性添加链接。

```html
<template>
  <v-breadcrumb>
    <v-breadcrumb-item to="/">Home</v-breadcrumb-item>
    <v-breadcrumb-item to="/components/breadcrumb">Components</v-breadcrumb-item>
    <v-breadcrumb-item>Breadcrumb</v-breadcrumb-item>
  </v-breadcrumb>
</template>
```
:::
