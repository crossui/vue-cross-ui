> 带有图标的

:::demo 可自定义每项的内容，比如带有一个图标。

```html
<template>
  <v-breadcrumb>
        <v-breadcrumb-item to="/">
            <v-icon type="home"></v-icon> Home
        </v-breadcrumb-item>
        <v-breadcrumb-item to="/components/breadcrumb">
            <v-icon type="gold"></v-icon> Components
        </v-breadcrumb-item>
        <v-breadcrumb-item>
            <v-icon type="tag"></v-icon> Breadcrumb
        </v-breadcrumb-item>
    </v-breadcrumb>
</template>
```
:::
