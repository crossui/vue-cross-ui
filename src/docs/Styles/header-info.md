# 常用样式
  项目中经常用到CSS常用的样式


>常用

| 类名 | 说明
| --- | --- |
| clearfix | 清除浮动 |
| fl | 左浮动 |
| fr | 右浮动 |
| text-left | 居左 |
| text-right | 居右 |
| text-center | 居中 |
| notwrap | 省略号 |
| height- | 预设高度 包含（100, 120, 150, 200, 250, 300, 350, 400, 500） 单位px|
| margin- | 外补白 预设（5, 10, 15, 20, 25, 30, 35, 40, 50, 100, 200） 单位px|
| margin-top- | 外补白上 预设（5, 10, 15, 20, 25, 30, 35, 40, 50, 100, 200） 单位px |
| margin-bottom- | 外补白下 预设（5, 10, 15, 20, 25, 30, 35, 40, 50, 100, 200） 单位px |
| margin-left- | 外补白左 预设（5, 10, 15, 20, 25, 30, 35, 40, 50, 100, 200） 单位px |
| margin-right- | 外补白右 预设（5, 10, 15, 20, 25, 30, 35, 40, 50, 100, 200） 单位px |
| padding- | 内补白 预设（5, 10, 15, 20, 25, 30, 35, 40, 50, 100, 200） 单位px |
| padding-top- | 内补白上 预设（5, 10, 15, 20, 25, 30, 35, 40, 50, 100, 200） 单位px |
| padding-bottom- | 内补白下 预设（5, 10, 15, 20, 25, 30, 35, 40, 50, 100, 200） 单位px |
| padding-left- | 内补白左 预设（5, 10, 15, 20, 25, 30, 35, 40, 50, 100, 200） 单位px |
| padding-right- | 内补白右 预设（5, 10, 15, 20, 25, 30, 35, 40, 50, 100, 200） 单位px |
